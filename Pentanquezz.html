Pentanquezz

Project Overview

Pentanquezz is a game designed for those who want to learn the sport of petanque in a fun and interactive way. In this game, players can access video tutorials to learn how to play petanque, as well as various minigames that test their knowledge of the sport. Wrapped in engaging gameplay, Pentanquezz offers an educational and enjoyable experience. The game is available to play on the Android platform.

Role in the Project

This is my first project, which starting by create image assets, designing the UI/UX, building the prototype, and implement it in Unity.
